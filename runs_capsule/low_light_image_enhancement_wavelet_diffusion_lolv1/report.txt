# Reproduction Report: Low-Light Image Enhancement with Wavelet-based Diffusion Models

## 1. Summary of the Paper's Approach

DiffLL is a diffusion-based framework for low-light image enhancement that combines wavelet transformation with diffusion models. The method uses a wavelet-based conditional diffusion model (WCDM) that performs diffusion operations on wavelet coefficients to reduce computational requirements while maintaining quality. Key innovations include:
- High-frequency restoration module (HFRM) for detail reconstruction
- Novel training strategy performing both forward diffusion and denoising during training for stable inference
- 70× faster than previous diffusion methods with 3× fewer computational resources

## 2. Methodology, Architecture, and Algorithms

### Core Approach:
- Uses 2D discrete wavelet transformation (2D-DWT) with Haar wavelets to convert images into wavelet domain
- Performs diffusion on the average coefficient (global information)
- Reconstructs high-frequency coefficients (vertical, horizontal, diagonal details) using HFRM

### Architecture Components:
- U-Net as noise estimator
- Depth-wise separable convolutions
- Cross-attention layers
- Progressive dilation ResBlocks

### Training Strategy:
- Diffusion loss
- Detail preserve loss (MSE + TV)
- Content loss (L1 + SSIM)

## 3. LOLv1 Dataset Specifications and Access

### Dataset Composition:
- 1,000 synthetic low/normal-light pairs

## 5. Evaluation Metrics for LOLv1

### Paired Datasets Metrics:
- **PSNR** (Peak Signal-to-Noise Ratio) - Higher is better
- **SSIM** (Structural Similarity Index) - Higher is better
- **LPIPS** (Learned Perceptual Image Patch Similarity) - Lower is better
- **FID** (Fréchet Inception Distance) - Lower is better

### Unpaired Datasets Metrics:
- **NIQE** (Natural Image Quality Evaluator) - Lower is better
